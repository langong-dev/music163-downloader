# Music163Downloader

# Install

Install [LanFilePackage](https://langong-dev.github.io/LanFilePackage/) first.

```bash
./LanGong install github langong-dev/music163-downloader
```

# Use

```bash
./LanGong run music163-downloader
```

Then input the link of your music.
